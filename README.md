# Online Retail RFM Analysis

This project focuses on customer segmentation using the RFM (Recency, Frequency, Monetary) model to analyze customer behavior and identify key segments for targeted marketing strategies.

## Project Overview
The objective is to transform raw transactional data into actionable insights by grouping customers based on their purchasing habits. This analysis helps in identifying high-value customers and those at risk of churning.

## Dataset
- **Source:** Online Retail II dataset.
- **Content:** Transactions of a UK-based online retail store between 2009 and 2011.

## Technologies Used
- **Python**: Main programming language.
- **Pandas**: For data cleaning and manipulation.
- **Matplotlib & Seaborn**: For data visualization.
- **Jupyter Notebook**: Development environment.

## Key Steps
1. **Data Preprocessing:** Cleaning missing values and removing canceled orders to ensure data integrity.
2. **Feature Engineering:** Calculating Recency, Frequency, and Monetary metrics for each customer.
3. **Customer Segmentation:** Assigning RFM scores and categorizing customers into segments such as Champions, Loyal Customers, and At-Risk.
4. **Visualization:** Creating charts to represent customer distribution and sales trends.

## Visualizations
The project includes several insights, such as:
- Customer Segmentation Distribution
- Top 10 Best Selling Products
- Total Revenue by Country

## How to Use
1. Clone the repository.
2. Install dependencies: `pip install pandas matplotlib seaborn`
3. Open and run `onlineretail.ipynb` in a Jupyter environment.
